🎨 Customization (SCSS)

1️⃣ Change Primary Colors

Open scss/_variables.scss and modify these values:

// _variables.scss - Your Gradient Colors
$primary-start: #7C3AED;    // rgb(124, 58, 237)
$primary-end: #8B5CF6;      // rgb(139, 92, 246)
$primary-gradient: linear-gradient(135deg, $primary-start, $primary-end);

// Other brand colors
$secondary: #10b981;         // Success green
$danger: #ef4444;            // Error red
$dark: #111827;              // Dark background
$light: #f9fafb;             // Light background
2️⃣ Compile SCSS to CSS

If you're using SCSS (recommended), compile with:

# Using command line
cd startupx-pro
sass --watch scss/style.scss:css/style.css

# Or use VS Code extension "Live Sass Compiler"
3️⃣ Direct CSS Override

Quick color change without SCSS (in css/style.css):

:root {
  --primary-start: #7C3AED;
  --primary-end: #8B5CF6;
  --primary-gradient: linear-gradient(135deg, var(--primary-start), var(--primary-end));
}

🌓 Dark Mode Support

StartUpX comes with built-in dark mode. Toggle by changing the data-bs-theme attribute on the HTML tag:

🌞 Light Mode (Default)
<html data-bs-theme="light">
  
</html>
🌙 Dark Mode
<html data-bs-theme="dark">
  
</html>
🔄 Dynamic Toggle JavaScript
function toggleTheme() {
  const html = document.documentElement;
  const current = html.getAttribute('data-bs-theme');
  const newTheme = current === 'light' ? 'dark' : 'light';
  
  html.setAttribute('data-bs-theme', newTheme);
  localStorage.setItem('theme', newTheme);
}

// Load saved preference
document.addEventListener('DOMContentLoaded', () => {
  const saved = localStorage.getItem('theme') || 'light';
  document.documentElement.setAttribute('data-bs-theme', saved);
});
